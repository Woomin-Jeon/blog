## Findings

- **margin 상쇄**  
  margin이 겹치게되면 두 margin 중 작은 margin이 상쇄되는 현상을 말합니다.

    ```css
    .a {
      margin-bottom: 20px;
    }

    .b {
      margin-top: 30px;
    }
    ```

    이 경우 a와 b의 거리는 50px이 아니라 상쇄되어 30px이 됩니다.
