## 성공

2020-03-11

```javascript
const solution = (a, b) => {
    let accumulation = 0;
    const smaller = a > b ? b : a;
    const bigger = a > b ? a : b;

    for (let i = smaller; i <= bigger; i++) {
        accumulation += i;
    }

    return accumulation;
}
```
