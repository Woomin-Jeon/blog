## 성공

2020-03-07

```javascript
const solution = (s) => {
  const splitedArray = s.split('');
  const pCount = splitedArray.filter(v => v == 'p').length;
  const PCount = splitedArray.filter(v => v == 'P').length;
  const yCount = splitedArray.filter(v => v == 'y').length;
  const YCount = splitedArray.filter(v => v == 'Y').length;
  
  return (pCount + PCount == yCount + YCount) ? true : false
}
```
