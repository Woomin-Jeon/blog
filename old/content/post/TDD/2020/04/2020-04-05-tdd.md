## 이해

- 자연수 n이 주어질 때 얘네들의 자릿수를 더한 값을 return한다.

## 계획

- toString을 이용해서 문자열로 변경한 뒤 배열로 나눈다음 reduce를 이용한다.

## 성공

2020-04-05

```javascript
const solution = (n) => n.toString().split('').map(v => +v).reduce((acc, cur) => acc + cur, 0);
```

## 회고

- 역시 toString을 이용하니 간단하게 풀 수 있었다.
