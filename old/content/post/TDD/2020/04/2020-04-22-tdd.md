## 이해

- 배열에서 제일 작은 값을 제거 한 후 return한다.

## 계획

- 배열에서 제일 작은 수를 찾은 후, 그 값의 index를 찾아서 제거한다.

## 성공

2020-00-00

```javascript
const solution = (arr) => {
  const tempt = arr.slice().sort((a, b) => a - b);
  arr.splice(arr.indexOf(tempt[0]), 1);
  return arr.length === 0 ? [-1] : arr;
}
```
