## 성공

2020-02-10

```javascript
const solution = (numbers) => {
  if (numbers.filter(v => v === 0).length === numbers.length) {
    return '0';
  }

  numbers.sort((a, b) => {
    return (
      +(a + '' + b) > + (b + '' + a) ? -1
      : +(a + '' + b) < + (b + '' + a) ? 1
      : 0
    );
  });

  return numbers.join('');
};
```
