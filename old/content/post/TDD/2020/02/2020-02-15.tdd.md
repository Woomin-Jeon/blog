## 성공

2020-02-15

```javascript
const solution = (n) => {
  if (Math.sqrt(n) === Math.floor(Math.sqrt(n))) {
    return (Math.sqrt(n) + 1) ** 2;
  }

  return -1;
};
```

## 리펙토링

2020-02-15

```javascript
const solution = (n) =>
  Math.sqrt(n) === Math.floor(Math.sqrt(n)) ? (Math.sqrt(n) + 1) ** 2 : -1;
```
