## 성공

2020-02-19

```javascript
const solution = (s) => {
    if (s.length % 2 === 0) {
        return `${s[s.length/2 -1]}${s[s.length/2]}`;
    } else {
        return s[Math.floor(s.length/2)];
    }
};
```

## 리펙토링

2020-02-19

```javascript
const solution = (s) =>
    s.length % 2 === 0
    ? `${s[s.length / 2 - 1]}${s[s.length / 2]}`
    : s[Math.floor(s.length/2)];
```
