## 성공

2020-02-26

```javascript
const solution = (arr) => {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] == arr[i+1]) {
      arr[i] += 100;
    }
  }

  arr = arr.filter(v => v < 100)

  return arr;
};
```

## 리펙토링

2020-02-26

```javascript
const solution = (arr) =>
  arr.map((v, i) => arr[i] == arr[i+1] ? arr[i] += 100 : v ).filter(v => v < 100);
```
