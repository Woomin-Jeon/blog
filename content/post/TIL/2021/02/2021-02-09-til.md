## Findings

- ES6이후의 forEach와 같이 순회하는 것들은 복사본을 돌립니다. 따라서 다음은 문제가 생기지 않습니다.

    ```js
    const arr = [1, 2, 3, 4];

    arr.forEach((v, i) => {
      arr.splice(i, 1);
    });
    ```
